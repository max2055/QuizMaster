# 打包发布指南

## 📦 打包说明

本指南介绍如何将 QuizMaster 打包为可执行文件，方便在没有 Python 环境的电脑上运行。

---

## 🖥️ Windows 打包

### 环境准备

1. 安装 Python 3.9+
   - 下载地址：https://www.python.org/downloads/
   - 安装时勾选 "Add Python to PATH"

2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```

### 打包步骤

#### 方法一：使用批处理脚本（推荐）

1. 双击运行 `build.bat`
2. 等待打包完成
3. 在 `dist` 文件夹中找到 `QuizMaster.exe`

#### 方法二：手动打包

```bash
# 清理旧文件
rmdir /s /q build dist

# 使用 PyInstaller 打包
pyinstaller --clean QuizMaster.spec

# 或使用命令行参数
pyinstaller --onefile --windowed --name QuizMaster --icon=resources/icons/app.ico main.py
```

### 打包参数说明

```
--onefile          # 打包为单个 exe 文件
--windowed         # 不显示控制台窗口
--name QuizMaster  # 指定输出文件名
--icon app.ico     # 指定应用图标
--clean            # 清理临时文件
```

### 输出文件

打包完成后，`dist` 文件夹中会生成：
- `QuizMaster.exe` - 主程序（约 50-100MB）

## 🍎 macOS 打包

### 环境准备

```bash
# 安装 Python 3.9+
brew install python@3.9

# 安装依赖
pip3 install -r requirements.txt
```

### 打包步骤

```bash
# 赋予脚本执行权限
chmod +x build.sh

# 运行打包脚本
./build.sh
```

### 输出文件

- `dist/QuizMaster` - macOS 可执行文件
- `dist/QuizMaster.app` - macOS 应用包（如配置）

### 签名和公证（可选）

为了在其他 Mac 上运行，需要签名：

```bash
# 代码签名
codesign --deep --force --verify --verbose --sign "Developer ID Application: Your Name" dist/QuizMaster.app

# 提交公证
xcrun notarytool submit dist/QuizMaster.app --apple-id "your@email.com" --password "app-password" --team-id "TEAM_ID"
```

## 🐧 Linux 打包

### 环境准备

```bash
# Ubuntu/Debian
sudo apt update
sudo apt install python3 python3-pip

# 安装依赖
pip3 install -r requirements.txt
```

### 打包步骤

```bash
chmod +x build.sh
./build.sh
```

### 输出文件

- `dist/QuizMaster` - Linux 可执行文件

### 创建桌面快捷方式

创建 `QuizMaster.desktop`：

```ini
[Desktop Entry]
Name=QuizMaster
Comment=刷题工具
Exec=/path/to/QuizMaster
Icon=/path/to/icon.png
Terminal=false
Type=Application
Categories=Education;
```

## 📱 分发建议

### Windows

1. **直接分发 EXE**
   - 将 `QuizMaster.exe` 压缩为 ZIP
   - 用户解压后直接运行

2. **创建安装程序**（可选）
   - 使用 Inno Setup
   - 使用 NSIS
   - 使用 WiX Toolset

### macOS

1. **DMG 分发**
   ```bash
   hdiutil create -volname "QuizMaster" -srcfolder dist/QuizMaster.app -ov -format UDZO QuizMaster.dmg
   ```

2. **App Store 发布**（需要开发者账号）

### Linux

1. **AppImage**（推荐）
   ```bash
   # 使用 linuxdeploy
   linuxdeploy --appdir AppDir -e dist/QuizMaster -i icon.png -d QuizMaster.desktop
   ```

2. **deb/rpm 包**
   - 使用 fpm 工具
   - 使用 dh-make

## 🔧 常见问题

### Q: 打包后文件太大？

A: 正常现象，PyInstaller 会打包 Python 解释器和所有依赖。可以使用以下方法优化：

```bash
# 使用 UPX 压缩（已在 spec 文件中启用）
--upx-dir=/path/to/upx

# 排除不需要的模块
--exclude-module matplotlib
```

### Q: 杀毒软件报毒？

A: PyInstaller 打包的程序可能被误报。解决方法：

1. 对程序进行数字签名
2. 向杀毒软件厂商提交白名单
3. 告知用户添加信任

### Q: 运行时报错找不到模块？

A: 在 spec 文件中添加 hiddenimports：

```python
hiddenimports=[
    'PyQt6.sip',
    'pandas',
    'openpyxl',
    # 添加其他隐式导入的模块
]
```

### Q: 如何添加应用图标？

A: 
1. 准备 `.ico` (Windows) 或 `.icns` (macOS) 图标文件
2. 放在 `resources/icons/` 目录
3. 在 spec 文件或命令行中指定 `--icon` 参数
