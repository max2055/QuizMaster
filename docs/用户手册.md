# QuizMaster 用户手册

## 📖 简介

QuizMaster 是一款功能明确、高效的题库管理和刷题工具，支持 Windows/macOS/Linux 跨平台运行。

### 主要功能

- ✅ **题库管理**：支持题目的添加、编辑、删除、批量导入导出
- ✅ **多种刷题模式**：顺序刷题、随机刷题、错题模式
- ✅ **分类管理**：树形结构支持多级分类
- ✅ **答题统计**：实时统计练习进度和正确率
- ✅ **错题本**：自动记录错题，支持标记掌握

---

## 🚀 快速开始

### 安装

#### 方法一：直接使用可执行文件（Windows）

1. 下载 `QuizMaster.exe`
2. 双击运行即可

#### 方法二：从源码运行

```bash
# 1. 克隆或下载项目
git clone <repository-url>
cd QuizMaster

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行程序
python main.py
```

### 首次使用

1. 启动程序后，左侧显示分类树
2. 点击"➕ 添加分类"创建第一个分类
3. 点击"➕ 添加题目"或"📥 导入"添加题目
4. 选择分类，点击"▶ 开始练习"开始刷题

---

## 📚 题库管理

### 添加分类

1. 点击左侧"➕ 添加分类"按钮
2. 输入分类名称
3. 选择父分类（可选，支持多级分类）
4. 点击"确定"

### 添加题目

1. 点击顶部"➕ 添加题目"按钮
2. 填写题目信息：
   - **题目内容**：支持多行文本
   - **所属分类**：选择题目所属分类
   - **题型**：单选题/多选题/判断题/填空题
   - **难度**：简单/中等/困难
   - **选项**：A-F（根据题型填写）
   - **正确答案**：例如 A 或 ABC
   - **答案解析**：可选
   - **标签**：用逗号分隔，例如：数学，代数
3. 点击"保存"

### 批量导入题目

支持从 Excel、CSV、Word 文件批量导入题目。

#### Excel/CSV 格式

| 题目 | 选项 A | 选项 B | 选项 C | 选项 D | 答案 | 解析 | 分类 | 难度 |
|------|--------|--------|--------|--------|------|------|------|------|
| 1+1 等于？ | 1 | 2 | 3 | 4 | B | 基础加法 | 数学 | 简单 |

**必填字段**：题目、答案  
**可选字段**：选项 A-F、解析、分类、难度、题型、标签

#### 导入步骤

1. 点击"文件" → "导入题目"或工具栏"📥 导入"
2. 选择 Excel/CSV/Word 文件
3. 点击"导入"
4. 等待导入完成

### 导出题库

1. 点击"文件" → "导出题库"
2. 选择保存位置和格式（CSV 或 Excel）
3. 点击"保存"

### 备份数据库

1. 点击"文件" → "备份数据库"
2. 选择保存位置
3. 点击"保存"

## 📝 刷题练习

### 选择练习模式

- **顺序刷题**：按题目 ID 顺序练习
- **随机刷题**：随机抽取题目
- **错题模式**：只练习错题本中的题目

### 开始练习

1. 在左侧选择要练习的分类（或选择"全部题目"）
2. 选择练习模式
3. 点击"▶ 开始练习"

### 答题界面

- **题目显示**：当前题目内容和选项
- **选项选择**：点击选项或使用键盘 1-6 选择
- **标记题目**：点击"📌 标记本题"标记疑难题目
- **提交答案**：点击"✓ 提交答案"或按 Enter 键
- **导航**：点击"上一题/下一题"或使用←→键

### 查看答案解析

提交答案后，会显示：
- ✅ 回答正确 / ❌ 回答错误
- 正确答案
- 答案解析（如果有）

### 错题本

答错的题目会自动加入错题本：
- 在"错题模式"下可以专门练习错题
- 答对多次后可以标记为"已掌握"
- 错题本统计在"统计分析"页面查看

## 📊 统计分析

### 总体统计

- 📚 总题量：题库中的题目总数
- ✏️ 已练习：已经练习过的题目数量
- 🎯 正确率：答题正确率百分比
- 📝 错题本：未掌握的错题数量

### 详细统计

- ✅ 正确数 / ❌ 错误数
- 💼 完成会话数
- 📊 练习进度条

### 刷新统计

点击"🔄 刷新统计"按钮更新最新数据

## 💡 使用技巧

1. **批量导入**：准备大量题目时，建议使用 Excel 批量导入
2. **分类管理**：合理分类可以提高刷题效率
3. **错题复习**：定期使用"错题模式"复习薄弱环节
4. **标记题目**：遇到难题先标记，后续重点复习
5. **定期备份**：重要题库记得定期备份数据库

## ❓ 常见问题

### Q: 导入的中文乱码怎么办？
A: CSV 文件请保存为 UTF-8 编码，Excel 文件通常无此问题。

### Q: 如何修改已添加的题目？
A: 目前版本暂不支持编辑，可以删除后重新添加。后续版本会添加编辑功能。

### Q: 数据库文件在哪里？
A: 默认在程序同目录下的 `quizmaster.db` 文件。

### Q: 可以多人使用吗？
A: 可以，但数据是共享的。如需独立数据，请复制数据库文件。
