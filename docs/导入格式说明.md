# 题目导入格式说明

## 📋 支持的格式

QuizMaster 支持从以下格式导入题目：
- Excel (.xlsx, .xls)
- CSV (.csv)
- Word (.docx, .doc)

---

## 📊 Excel/CSV 格式

### 标准格式

| 列名 | 必填 | 说明 | 示例 |
|------|------|------|------|
| 题目 | ✅ | 题目内容 | 1+1 等于？ |
| 选项 A | ❌ | 选项 A 内容 | 1 |
| 选项 B | ❌ | 选项 B 内容 | 2 |
| 选项 C | ❌ | 选项 C 内容 | 3 |
| 选项 D | ❌ | 选项 D 内容 | 4 |
| 选项 E | ❌ | 选项 E 内容 | （可选） |
| 选项 F | ❌ | 选项 F 内容 | （可选） |
| 答案 | ✅ | 正确答案 | B 或 ABC |
| 解析 | ❌ | 答案解析 | 基础加法运算 |
| 分类 | ❌ | 所属分类 | 数学 |
| 难度 | ❌ | 难度等级 | easy/medium/hard |
| 题型 | ❌ | 题目类型 | single/multi/true_false/fill |
| 标签 | ❌ | 题目标签 | 数学，代数，基础 |

### 示例数据

```
题目，选项 A，选项 B，选项 C，选项 D，答案，解析，分类，难度
1+1 等于？,1,2,3,4,B,基础加法，数学，easy
中国的首都是？，上海，北京，广州，深圳，B,常识题，地理，easy
以下哪些是质数？,2,3,4,5,ABD,质数定义，数学，medium
```

### 难度取值

- `easy` - 简单
- `medium` - 中等
- `hard` - 困难

### 题型取值

- `single` - 单选题
- `multi` - 多选题
- `true_false` - 判断题
- `fill` - 填空题

### 答案格式

- **单选题**：单个字母，如 `A`
- **多选题**：多个字母，如 `ABC` 或 `A,B,C`
- **判断题**：`正确 `/` 错误` 或 `T`/`F`
- **填空题**：填写内容，如 `北京`

## 📄 Word 格式

### 结构化格式

Word 文档支持以下结构：

```
1. 题目内容是什么？
A. 选项 A 内容
B. 选项 B 内容
C. 选项 C 内容
D. 选项 D 内容
答案：B
解析：这是答案解析

2. 第二道题目？
A. 选项 A
B. 选项 B
答案：A
分类：数学
难度：easy
```

### 格式要求

- 题目以数字开头（如 `1.` `2.` 或 `第 1 题`）
- 选项以字母开头（如 `A.` `B.` 或 `A、`）
- 答案以 `答案：` 或 `正确答案：` 开头
- 解析以 `解析：` 或 `答案解析：` 开头
- 分类以 `分类：` 开头
- 难度以 `难度：` 开头

## 💡 最佳实践

### 1. 使用 Excel 批量导入

Excel 是最推荐的导入方式，因为：
- 易于编辑和整理
- 支持批量操作
- 格式清晰直观

### 2. 分类命名规范

建议使用清晰的分类层级：
```
数学
├── 代数
├── 几何
└── 概率

英语
├── 词汇
├── 语法
└── 阅读
```

### 3. 标签使用

使用逗号分隔的标签便于后续搜索：
```
数学，代数，一元二次方程，基础
英语，语法，时态，现在完成时
```

### 4. 难度标注

根据实际难度标注：
- **easy**: 基础题，大部分人都能做对
- **medium**: 中等难度，需要思考
- **hard**: 困难题，容易出错

---

## ⚠️ 注意事项

1. **编码问题**：CSV 文件请保存为 UTF-8 编码
2. **必填字段**：题目和答案是必填项
3. **选项数量**：至少需要 2 个选项
4. **答案格式**：多选题答案按字母顺序排列（如 ABC 而非 CBA）
5. **特殊字符**：避免在题目中使用 Excel 特殊字符（如`=`, `+` 开头的内容需加单引号）

## 🔧 模板下载

可以创建以下模板文件作为参考：

### Excel 模板

创建 `quiz_template.xlsx`，包含以下列：
```
题目 | 选项 A | 选项 B | 选项 C | 选项 D | 答案 | 解析 | 分类 | 难度 | 题型 | 标签
```

### CSV 模板

创建 `quiz_template.csv`：
```csv
题目，选项 A，选项 B，选项 C，选项 D，答案，解析，分类，难度，题型，标签
示例题目，选项 A 内容，选项 B 内容，选项 C 内容，选项 D 内容，B,解析内容，分类名，medium,single，标签 1，标签 2
```

## 📞 问题反馈

如遇到导入问题，请检查：
1. 文件格式是否正确
2. 列名是否匹配
3. 文件编码是否为 UTF-8
4. 必填字段是否完整
