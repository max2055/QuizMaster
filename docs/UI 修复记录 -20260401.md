## 🔧 修复内容

### 1.3.3 修复（题型过滤功能）

#### 题型过滤功能
**问题**: 用户在左侧题库选择"单选题"或"多选题"后，开始练习没有按题型过滤

**修复**:
- 在 `QuestionWidget` 中添加 `question_type_filter` 属性
- 在控制栏下方添加题型选择按钮（全部/单选/多选/判断/填空）
- 修改 `load_questions` 方法，根据 `question_type_filter` 过滤题目
- 随机模式添加 `random.shuffle()` 打乱题目顺序

**新增元素**:
- 题型选择按钮组（5 个按钮，分段样式）
- 按钮尺寸：50×26px，字体 9px
- 布局位置：搜索框左侧，与搜索框之间加分隔线

#### 错题模式显示上次答案
**问题**: 错题模式下，用户想知道上次答的是什么

**修复**:
- 在 `_show_question` 方法中，错题模式下调用 `practice_service.get_question_record()`
- 显示"上次答案：X | 正确答案：Y"格式
- 颜色：橙黄色 (#F9AB00)，易于区分

**代码修改**:
```python
# 错题模式：显示上次答案
if self.practice_mode == 'wrong' and self.current_question:
    last_record = self.practice_service.get_question_record(self.current_question.id)
    if last_record and last_record.user_answer:
        self.answer_status_label.setText(
            f"上次答案：{last_record.user_answer} | 正确答案：{self.current_question.answer.upper()}"
        )
        self.answer_status_label.setStyleSheet("color: #F9AB00; font-weight: bold;")
```
