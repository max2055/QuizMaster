## 📦 打包步骤（开发者操作）

### 步骤 1：打包程序
```
双击运行：package_for_windows.bat
```
等待 1-3 分钟，生成 `dist/QuizMaster.exe`

### 步骤 2：准备分发文件
```
双击运行：prepare_distribution.bat
```
自动创建 `QuizMaster_分发版` 文件夹，包含：
- QuizMaster.exe
- quizmaster.db（已清空）
- README_用户说明.txt

### 步骤 3：压缩分发
1. 右键 `QuizMaster_分发版` 文件夹
2. 发送到 → 压缩文件夹
3. 生成 `QuizMaster_分发版.zip`
4. 发送给用户

---

## 📖 用户使用流程

1. 用户下载并解压 zip 文件
2. 双击 `QuizMaster.exe` 运行
3. 无需安装 Python 或其他环境
