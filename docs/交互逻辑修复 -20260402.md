```python
    clean_name = category_name.replace('📚', '').replace('📖', '').strip()
    if '单选' in clean_name:
        question_type_filter = 'single'
    elif '多选' in clean_name:
        question_type_filter = 'multi'
    elif '判断' in clean_name:
        question_type_filter = 'true_false'
    elif '填空' in clean_name:

if question_type_filter:

        if q.question_type in allowed_types
    ]
```

**修改内容**: 将 `_start_practice` 拆分为两个方法

```python
def _start_practice(self):
    """开始练习 - 带二次确认"""
    if not self.questions:
        return

    # 检查是否已答题，如果有则显示确认对话框
    if self.user_answers:
        from PyQt6.QtWidgets import QMessageBox
        reply = QMessageBox.question(
            self,
            "确认重新开始",
            f"当前已有 {len(self.user_answers)} 道题的答案，重新开始将清空所有进度。\n\n确定要重新开始吗？",
            QMessageBox.StandardButton.Yes | QMessageBox.StandardButton.No
        )

        if reply != QMessageBox.StandardButton.Yes:
            return

    self._start_practice_internal()

def _start_practice_internal(self):
    """内部开始练习方法（无确认）"""
    # ... 原有的开始练习逻辑
```

**显示"重新开始"的时机**:

- 即：**只要开始练习后**，按钮就一直显示"重新开始"

**二次确认触发条件**:

## 🎯 使用指南

### 题型过滤
1. 在左侧分类树中选择"📚 单选题"分类
2. 点击"顺序"或"随机" → "开始练习"
3. 此时练习的题目**全部为单选题**

### 题目编号
- 练习时题目显示格式：`[ID] 题目内容`
- 答题卡显示的是数据库原始 ID
- 结果汇总中也显示原始 ID

### 进度延续
1. 打开设置（Ctrl+,）
2. 勾选"启动时继续上次刷题进度"
3. 完成一批题目后，进度自动保存
4. 下次启动应用时自动恢复进度

### 重新开始
- 首次点击分类自动开始练习
- 练习中点击"重新开始"按钮：
  - 如果已有答题 → 弹出确认对话框
  - 如果未答题 → 直接重新开始

## ⚠️ 注意事项

1. **题型过滤依赖分类名称**：分类名称中需要包含"单选"、"多选"、"判断"、"填空"等关键词
2. **原始 ID 显示**：导入的题目会自动分配 ID，练习时显示的就是这个 ID
3. **进度保存**：只有完成一批题目或切换批次时才会保存，中途退出可能丢失进度
4. **重新开始确认**：只有已有答题时才会确认，避免误操作丢失进度
