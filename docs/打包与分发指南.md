# QuizMaster 打包与分发指南

## 📦 打包步骤（开发者操作）

### 方法一：一键打包脚本（推荐）

1. **双击运行** `package_for_windows.bat`

2. 等待打包完成（约 1-3 分钟）

3. 打包完成后，可执行文件位于：
   ```
   dist/QuizMaster.exe
   ```

### 方法二：手动打包

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 安装 PyInstaller
pip install pyinstaller

# 3. 打包
pyinstaller --clean QuizMaster.spec
```

---

## 📤 分发给用户

### 方案 A：直接发送 exe（简单，单文件）

**优点**：
- 用户只需一个文件，双击即可运行
- 无需安装任何环境

**缺点**：
- 文件较大（约 30-50 MB）

**操作**：
1. 将 `dist/QuizMaster.exe` 发送给用户
2. 用户双击即可运行

---

### 方案 B：打包成压缩包（推荐）

**优点**：
- 包含数据库和资源文件
- 更规范，便于更新维护

**操作**：

1. **创建分发文件夹**
   ```
   QuizMaster/
   ├── QuizMaster.exe          # 主程序
   ├── quizmaster.db           # 数据库（已清空）
   └── README_用户说明.txt     # 使用说明
   ```

2. **压缩成 zip**
   - 右键 → 发送到 → 压缩文件夹
   - 生成 `QuizMaster_v1.0.zip`

3. **发送给用户**
   - 通过微信/QQ/邮箱发送 zip 文件

```
=================================================
```

## 🔧 常见问题

### Q1: 杀毒软件报毒？
**A**: PyInstaller 打包的 exe 可能被误报。解决方法：
- 添加到杀毒软件白名单
- 或向杀毒软件厂商提交误报

### Q2: 双击无反应？
**A**: 可能是缺少 Visual C++ 运行库
- 下载地址：https://aka.ms/vs/17/release/vc_redist.x64.exe

### Q3: 如何更新题库？
**A**: 
1. 在开发者电脑上更新题库
2. 重新打包
3. 发送新的 quizmaster.db 给用户替换
