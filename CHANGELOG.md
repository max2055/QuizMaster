# QuizMaster 更新日志

## [1.2.0] - 2026-03-29

### ✨ 新增功能

#### 标记持久化
- 标记状态保存到数据库，重启后不丢失
- `PracticeService` 添加 `get_marked_questions()` 和 `update_question_mark()` 方法
- 加载题目时自动恢复标记状态

#### 判断题 UI
- 新增专用判断题界面，显示"✓ 正确"和"✗ 错误"两个大按钮
- 按钮支持单选，选中后蓝色高亮
- 答案格式：`"正确"` 或 `"错误"`

#### 填空题 UI
- 新增填空题专用输入框（QTextEdit）
- 支持多行文本输入，最小高度 100px
- 实时保存用户输入到答题记录

#### 自动下一题
- 添加"⏭ 自动下一题"开关按钮
- 答题后 0.5 秒自动跳转到下一题
- 最后一题答题后自动提示提交

#### 错题掌握标记
- 错题模式添加"✅ 已掌握"按钮
- 点击可标记/取消标记题目为已掌握
- 已掌握题目可从错题本过滤

### 🎨 UI 优化

#### 判断题按钮样式
- 60px 高度，12px 圆角
- 3px 边框，悬停时蓝色高亮
- 选中状态：蓝色背景 + 白色文字

#### 填空题输入框样式
- 2px 边框，8px 圆角
- 12px 内边距，白色背景
- 聚焦时自动调整高度

### 🔧 技术优化

- 所有 v1.2.0 功能测试通过
- 代码符合 PEP 8 规范
- 完整的类型注解

## [1.1.0] - 2026-03-28

### ✨ 新增功能

#### 📋 答题卡侧边栏
- 右侧新增答题卡视图，显示所有题目的答题状态
- 状态颜色：灰色=未作答，蓝色=已选择，绿色=正确，红色=错误
- 点击题号可快速跳转到对应题目
- 显示图例说明和当前题目高亮

#### 🔍 搜索功能
- 工具栏添加搜索框，支持搜索题目
- 预留搜索接口，可扩展按题目内容、分类、标签搜索

#### ✏️ 编辑/删除题目
- 刷题界面添加"编辑"和"删除"按钮
- 支持在当前题目界面直接编辑和删除题目
- 分类树支持右键菜单编辑/删除分类

#### 📤 导出功能完善
- 实现导出为 CSV 格式
- 实现导出为 Excel 格式（需要 pandas + openpyxl）
- 支持导出当前分类或全部题目

#### ⌨️ 快捷键支持
- **1-6**: 选择选项 A-F
- **←/→**: 上一题/下一题
- **Enter**: 下一题（答题后）
- **Shift+Enter**: 提交答案
- **Ctrl+S**: 编辑当前题目
- **Ctrl+D**: 删除当前题目
- **Ctrl+Enter**: 提交所有答案
- **空格**: 标记/取消标记题目

#### 🔀 多选题支持
- 根据题型动态切换 UI（单选=RadioButton, 多选=CheckBox）
- 多选题支持选择多个选项
- 提交时自动判断答案是否完全匹配
- 答题卡支持显示多选题答题状态

### 🎨 改进

#### UI 优化
- 工具栏添加导出按钮
- 搜索框集成到工具栏
- 编辑/删除按钮集成到刷题界面
- 分类树支持右键菜单

#### 交互优化
- 答题后自动更新答题卡状态
- 提交答案后显示完整汇总
- 题目跳转功能完善

### 🐛 修复
- 修复部分布局按钮被挤压问题
- 修复选项文本重叠问题

## [1.0.4] - 2026-03-26

### 🎨 UI 改进 - macOS 窗口标题栏

- **修复 macOS 窗口标题栏深色问题**
  - 添加 `Info.plist` 配置文件
  - 设置 `NSRequiresAquaSystemAppearance` 强制浅色模式
  - 更新 PyInstaller spec 文件包含 Info.plist 配置
  
- **优化 macOS 调色板**
  - 设置完整的浅色 QPalette
  - 统一所有 UI 组件颜色
  - 优化窗口和文本颜色对比

- **注意**: macOS 系统级深色模式会影响标题栏颜色
  - 开发模式运行：标题栏可能仍为深色（系统限制）
  - 打包后运行：标题栏将强制为浅色
  - 建议：打包后体验完整浅色主题

### 打包说明

**macOS 打包后强制浅色模式**:
```bash
./build.sh
# 生成的应用将使用浅色标题栏
```

**临时解决方案** (开发模式):
在系统设置中临时切换为浅色模式：
```
系统设置 → 外观 → 浅色
```
