# CLAUDE.md

## 项目概述
QuizMaster - 题库管理和组卷工具

## Auto Mode 配置

### 模式设置
- **Auto Mode 已启用** - 使用 `/auto` 命令激活
- AI 可以自主规划和执行多步骤任务
- 无需逐条确认即可进行常规开发工作

### Auto Mode 行为准则
- **小型改动**（单文件或少量代码）：直接执行
- **中型改动**（多文件修改）：先简要说明计划，然后执行
- **大型改动**（架构调整/重大变更）：先征求意见再执行
- **破坏性操作**（删除文件/数据）：必须明确确认

## 开发规范

### 权限模式
- **Bypass permissions 模式已启用**
- 可以直接执行文件操作和代码修改
- 无需每次确认即可进行常规开发任务

### 编码风格
- 使用 4 个空格缩进
- 遵循 PEP 8 规范
- 函数和类需要类型注解

### 测试
- 运行测试：`python test_app.py`
- 修改代码后确保测试通过

### 构建
- Windows: `build.bat`
- macOS/Linux: `build.sh`
